# CRO Experiment Library

A curated collection of conversion-rate-optimization test ideas for SaaS and ecommerce teams. Built by [CROLabs](https://crolabs.com).

## Pages

- **Landing Page Tests** — 12 experiments for hero copy, CTAs, proof blocks, forms, and layout
- **Pricing Page Tests** — 10 experiments for toggles, plan emphasis, trust signals, and feature tables
- **Signup & Demo Page Tests** — 10 experiments for form length, social proof, incentive framing, and objections
- **A/B Test Brief Template** — Interactive form to document experiments before launch
- **Sample-Size Calculator** — Estimate visitors needed per variant for statistical significance

## Setup

This is a static HTML site designed to be deployed via GitHub Pages.

1. Push this repo to GitHub as a **public** repository
2. Go to **Settings > Pages**
3. Under **Build and deployment**, select **Deploy from a branch**
4. Set branch to **main** and folder to **/ (root)**
5. Save — the site will be live within a minute

## Local preview

Open `index.html` in a browser. No build step required.
